Configuração inicial
Antes de usar o painel, configura o Supabase em 3 passos. Só precisas de fazer isto uma vez.
1
Cria um projeto novo em supabase.com e copia a URL e a anon key.
2
No SQL Editor do Supabase, corre o script abaixo para criar a tabela e o bucket de fotos.
3
Cola a URL e a anon key nos campos abaixo e clica em Guardar e entrar.
-- Cola isto no SQL Editor do Supabase: create table veiculos ( id uuid primary key default gen_random_uuid(), ordem integer not null default 0, tipo text not null default 'moto', -- 'moto' | 'carro' nome text not null, marca text, ano integer, motor text, potencia text, peso text, tag text, descricao text, fotos jsonb default '[]'::jsonb, criado_em timestamptz default now() ); -- Bucket público para fotos: insert into storage.buckets (id, name, public) values ('veiculos-fotos', 'veiculos-fotos', true); -- Política de upload (service_role ou auth): create policy "Public read" on storage.objects for select using (bucket_id = 'veiculos-fotos'); create policy "Auth upload" on storage.objects for insert with check (bucket_id = 'veiculos-fotos');
